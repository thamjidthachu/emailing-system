# Introduction

## Requirements

 - Python version `3.9`
 - Django version `3.2`
 - Pipenv



# Steps to run the project

## Install packages

````bash
    pip install .requirements.txt
````

## prepare and runserver
```bash
    ./manage.py migrate
    ./manage.py runserver
```

## create superuser
```bash
    ./manage.py createsuperuser
```
